Как загрузить сайт на GitHub Pages

1. Открой свой репозиторий на GitHub.
2. Нажми Add file -> Upload files.
3. Загрузите файл index.html из этой папки.
4. Нажми Commit changes.
5. Зайди в Settings -> Pages.
6. В Source выбери Deploy from a branch.
7. В Branch выбери main и папку /(root).
8. Нажми Save.
9. Подожди 1-3 минуты.
10. Ссылка будет в формате:
   https://ТВОЙ-ЛОГИН.github.io/ИМЯ-РЕПОЗИТОРИЯ/

Важно:
- Счётчики сохраняются в браузере телефона автоматически.
- Используй один и тот же браузер на телефоне.
- Если очистить данные сайта или браузера, значения могут сброситься.
